발자국일기란?

장소를 옮길 때마다 자동으로 발자국이 쌓이고, 앱을 열면 "여기는 어디였나요?"라고 물어봐 주는 iOS 위치 일기 앱입니다. 하루의 이동 경로를 지도에 발자국으로 보여주고, 날짜별로 사진과 함께 일기를 쓸 수 있습니다.

  • 자동 발자국 기록 — 방문 감지로 배터리를 거의 쓰지 않고 이동을 기록
  • 지도 발자국 — 날짜별 이동 경로를 번호 붙은 👣 마커로 표시
  • 사진 일기 — 날짜별로 사진과 함께 일기 작성
  • 기기 내 저장 — 모든 데이터는 기기 안에만 저장, 외부 서버 전송 없음
