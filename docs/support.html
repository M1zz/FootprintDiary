자주 묻는 질문 (FAQ)

발자국이 자동으로 기록되지 않아요.

자동 기록은 iOS의 위치 권한이 "항상 허용"으로 설정되어 있어야 동작합니다. 설정 앱 → 발자국일기 → 위치 에서 "항상"을 선택해 주세요. 또한 방문 감지는 일정 시간 한 장소에 머문 뒤 이동해야 기록되므로, 즉시 반영되지 않을 수 있습니다. 바로 남기고 싶다면 지도 화면 오른쪽 위 ➕ 버튼을 사용하세요.

내 위치 정보가 외부로 전송되나요?

아니요. 모든 발자국·일기·사진은 기기 안에만 저장되며 개발자 서버로 전송되지 않습니다. 단, 좌표를 주소 이름으로 바꾸기 위해 Apple의 지도/지오코딩 서비스가 사용됩니다. 자세한 내용은 개인정보 처리방침을 확인하세요.

차량으로 이동한 경로도 기록되나요?

앱은 동작(모션) 데이터를 참고해 도보·뛰기 위주의 이동을 기록하고, 차량 이동으로 판단되는 구간은 발자국에서 제외하려고 합니다. 상황에 따라 일부 오차가 있을 수 있습니다.

기기를 바꾸면 데이터가 옮겨지나요?

데이터는 기기 내부에 저장되므로, 앱을 삭제하거나 기기를 초기화하면 데이터도 사라집니다. 현재 별도의 클라우드 백업/동기화 기능은 제공하지 않습니다.

발자국이나 일기를 삭제하고 싶어요.

지도 화면에서 발자국 마커나 목록을 탭해 개별 삭제할 수 있고, 일기는 일기 화면에서 삭제할 수 있습니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.
